== Description ==
WordPress is a monolith of plugins and customizations that can overwhelm even the most experienced power user. 
Let us help you make the right decisions to maximize the power and efficiency of your WordPress site.
This build has been battle tested in the industry for flexability, speed, & ease of use. Our methodology is to build everything
with Elementor and Funnelmentals and not rely on themes at all.

= What will this plugin do for you? =
* Install a customize build out for WordPress in under 5 minutes with one click.

= What is the end result =
* All Elements on the page can be visually edited. Drag, drop, and rescale everything!
* Full advantage of tablet & mobile responsiveness.
* Header, footer, and templates will be fully customizable.
* Full support for metadata with the search engines & social networks.
* You will have a custom hyper-minimalistic theme.
* Unlimited proffesionally design prefab pages and sections.
* We only install plugins we trust.



###More Information:
Get more information and view the documentation at:
[webdisrupt.com](https://webdisrupt.com/wp-assistant).

== Installation ==
1. Upload **"web-disrupt-wp-assistant.zip"** to the **"/wp-content/plugins/"** directory.
2. Activate the plugin through the **"Plugins"** menu in WordPress.
